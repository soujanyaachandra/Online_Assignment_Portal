Guidelines to Online Test

  1. Stable Internet Connectivity is a must.
  2. Giving the test on a Laptop or Desktop is recommended.
  3. It is recommended to use web browsers such as Chrome, Mozilla, etc.
  4. You will have a total of 15 minutes to complete the test.
  5. After logging in, you must enter your name to display on the test screen.
  6. When the test screen opens, you can find the questions that you need to answer.
  7. On the right side, you can see the question numbers.
  8. Using those numbers, you can navigate between the questions.
  9. The different colors added to each question number are explained below:
    • Unvisited Question
    • Current Question
    • Visited but not Answered
    • Answered
    • Mark for Review
  10. On top, you can see the progress bar which displays the number of questions answered by you.
  11. You can also see the timer which starts running when the test begins.
  12. You can submit the test on your own by clicking the Submit button and see your score.

  14. The pass percentage for the test is at least 70%.
  15. You must get a 70% score to pass the test, or it will be considered a fail.
